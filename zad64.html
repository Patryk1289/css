1. :firstline to pseudoelement który obejmuje formatowaniem całą pierwszą linię
    Przykład:
    p
    {
    color:navy;
    font-size:12pt;
    }
    p:first-line
    {
    color:red;
    font-size:20pt;
    font-weight:bold;
    }
  2. :after oraz :before Oba elementy służą do umieszczania tekstu przed i za formatowanym elementem strony.
    Przykład:
    p:before
    {
    content:"TO SĄ ĆWICZENIA";
    font-size:15pt;
    font-style:italic;
    font-weight:bold;
    }
    p:after
    {
    content:"AUTOR Król Lew";
    font-size:15pt;
    font-style:italic;
    font-weight:bold;
    }
  3. :first-letterpozwala na formatowanie pierwszej litery w bloku tekstu.
    Przykład:
    p
    {
    color:navy;
    font-size:30pt;
    }
    p:first-letter
    {
    color:red;
    font-size:100pt;
    }
